# Checkout App
<img src="public/assets/checkout.png">

This is a simple checkout application built with Vue 3 + Vite for the frontend and TailwindCss for the styling.

## Prerequisite
1. HTML
2. TailwindCss
3. Vue 3
## Installation

```
$ git clone https://github.com/judesafe/Checkout-app.git
```

````
$ npm install
````
